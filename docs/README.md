# IoT Platform Documentation

Site de documentation officiel construit avec [Docusaurus](https://docusaurus.io/).

## Développement

```bash
make docs-dev  # Lance le serveur sur http://localhost:3000
```

## Build

```bash
make docs-build  # Génère le site statique dans build/
```
